DB processing:
    1) load sestoji layer into QGIS from WFS in Pregledovalnik
    2) we use REFACTOR from the toolbox to isolate sestoji identifiers
    3) we use JOIN ATTR. BY LOCATION from toolbox on the identifiers layer (touches)
    4) in sqlite browser we open the db with the joined layer and create new nodes, edges, isolate nodes, tables (remember to delete self edges!)
